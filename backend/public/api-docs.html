API Documentation

Welcome to the NovaEdge Developer Platform. Our REST API allows you to integrate our powerful digital tools directly into your own applications, websites, and internal workflows.

Authentication

Every request to the NovaEdge API must include your API Key in the x-api-key header. You can find and manage your key in the Developer API section of your Business Profile in the NovaEdge app.

x-api-key: your_key_here

Rate Limits & Quota

Business plan users receive a monthly quota of 5,000 API calls. This quota resets on the first of every month. If you exceed this limit, requests will return a 429 Too Many Requests status code.

QR Code Generator

Generate high-quality QR codes for any text or URL.

POST /api/v1/tools/generate-qr
Request Body
ParameterDescription
textREQUIREDstringThe content to encode in the QR code.
sizeintegerSize of the image in pixels (default: 300).
colorstringHex code for the foreground (default: #000000).
Example Response
{
  "success": true,
  "qrBase64": "data:image/png;base64,iVBORw0K...",
  "size": 300
}

Image Compression

Automatically optimize images for web and mobile use.

POST /api/v1/tools/compress-image
Request Body (Multipart)
FieldDescription
imageREQUIREDfileThe image file to compress (JPEG, PNG).
Example Response
{
  "success": true,
  "originalSize": 2540120,
  "compressedSize": 450120,
  "url": "https://cloudinary.com/..."
}

GST Calculator

Fast calculations for GST inclusive and exclusive pricing.

POST /api/v1/tools/calculate-gst
Request Body
ParameterDescription
amountREQUIREDnumberThe base amount.
gstRateREQUIREDintegerPercentage rate (e.g., 18).
